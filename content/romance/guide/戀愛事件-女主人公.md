# 戀愛事件攻略（女主人公）

本篇攻略說明女主人公追求 6 位可攻略角色的戀愛事件觸發條件與求婚方式。可攻略角色為：亞修（アーシュ）、卡米爾（カミル）、奇利克（キリク）、千尋（チヒロ）、米海爾（ミハイル）、迪魯卡（ディルカ）。

## 愛情度花朵顏色系統

可攻略角色的愛情度以花朵「顏色」表示，共分 7 個等級，與一般村民的好感度花朵「數量」系統不同：

| 等級 | 花朵顏色 |
|------|----------|
| Lv.1 | 白色 |
| Lv.2 | 紫色 |
| Lv.3 | 藍色 |
| Lv.4 | 綠色 |
| Lv.5 | 橙色 |
| Lv.6 | 粉色花苞（5 朵＋花苞） |
| Lv.7 | 大紅花（5 朵＋大花） |

> 戀愛事件需依 1→2→3→4 順序觸發。節日活動當天無法觸發約會事件。

![愛情度花朵等級示意圖](../images/leomoon173-pixnet-5010168176/52daf858b08640e210d7a2b5c15de159.jpg)

## 通用求婚說明

- **青之羽毛（青い羽根）**：愛情度達 50,000（粉色花苞）以上後，第 2 年起雜貨店開始販售。向對象出示即可求婚。
- **雙人床（ダブルベッド）**：所有角色的求婚前置條件之一，需透過自宅增建取得。
- **婚禮時間**：求婚成功後一週舉行婚禮。若婚禮當天遇到節日或村民生日，婚禮延至翌日（兩村的節日皆有影響）。
- **逆求婚**：奇利克（キリク）與千尋（チヒロ）可由女主人公主動持青之羽毛求婚，無需等待對方。

### 迪魯卡（ディルカ）

![迪魯卡 相關圖片](../images/leomoon173-pixnet-5010168176/48b66c20b898472f67b62e1b2f5dde8f.jpg)

**出現條件**：第 1 年夏以後，晴天 AM10:00～PM18:59 從山道進入。

**特殊規則**：
- 迪魯卡沒有戀愛事件，花朵顏色不會隨愛情度等級改變。
- 好感度需透過每日互動（送禮、約會）逐步累積，無捷徑。

#### 求婚條件（共 6 項，須**全部**達成）

1. 愛情度 65,000 以上（5 朵花＋大花）
2. 擁有雙人床（ダブルベッド）
3. 隧道開通
4. 與迪魯卡約會 **50 次以上**
5. 送迪魯卡禮物 **80 次以上**
6. 送過奶油炸肉餅（クリームコロッケ）**至少 1 次**

**求婚方式**：達成所有條件後，任意日期（晴天或雪天），持青之羽毛（青い羽根）向迪魯卡求婚。
