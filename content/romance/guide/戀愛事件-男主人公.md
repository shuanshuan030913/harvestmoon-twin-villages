# 戀愛事件攻略（男主人公）

本篇說明男主人公追求可攻略角色共通的戀愛事件機制——愛情度花朵等級、求婚方式、共通道具。
個別角色（莉亞、拉茲貝莉、莉可麗絲、娜娜、賢者大人、艾瑞拉）各自的戀愛事件觸發條件、對話選項與求婚細節，
已收錄在各角色自己的條目頁「戀愛事件」段落（2026-07-19 使用者裁決：條目頁以角色為單位收整明細，
本篇只留跨角色的通用規則）。

## 愛情度花朵顏色系統

可攻略角色的愛情度以花朵「顏色」表示，共分 7 個等級，與一般村民的好感度花朵「數量」系統不同：

| 等級 | 花朵顏色 |
|------|----------|
| Lv.1 | 白色 |
| Lv.2 | 紫色 |
| Lv.3 | 藍色 |
| Lv.4 | 綠色 |
| Lv.5 | 橙色 |
| Lv.6 | 粉色花苞（5 朵＋花苞） |
| Lv.7 | 大紅花（5 朵＋大花） |

> 戀愛事件需依 1→2→3→4 順序觸發。節日活動當天無法觸發約會事件。

![愛情度花朵等級示意圖](../images/leomoon173-pixnet-5010160702/flower3.jpg)

## 通用求婚說明

- **青之羽毛（青い羽根）**：愛情度達 50,000（粉色花苞）以上後，第 2 年起雜貨店開始販售。向對象出示即可求婚。
- **雙人床（ダブルベッド）**：所有角色的求婚前置條件之一，需透過自宅增建取得。
- **婚禮時間**：求婚成功後一週舉行婚禮。若婚禮當天遇到節日或村民生日，婚禮延至翌日（兩村的節日皆有影響）。
