# 戀愛事件攻略（男主人公）

本篇攻略說明男主人公追求 6 位可攻略角色的戀愛事件觸發條件與求婚方式。可攻略角色為：莉亞（リア）、拉茲貝莉（ラズベリー）、莉可麗絲（リコリス）、娜娜（ナナ）、賢者大人（賢者さま）、艾瑞拉（アリエラ）。

## 愛情度花朵顏色系統

可攻略角色的愛情度以花朵「顏色」表示，共分 7 個等級，與一般村民的好感度花朵「數量」系統不同：

| 等級 | 花朵顏色 |
|------|----------|
| Lv.1 | 白色 |
| Lv.2 | 紫色 |
| Lv.3 | 藍色 |
| Lv.4 | 綠色 |
| Lv.5 | 橙色 |
| Lv.6 | 粉色花苞（5 朵＋花苞） |
| Lv.7 | 大紅花（5 朵＋大花） |

> 戀愛事件需依 1→2→3→4 順序觸發。節日活動當天無法觸發約會事件。

![愛情度花朵等級示意圖](../images/leomoon173-pixnet-5010160702/flower3.jpg)

## 通用求婚說明

- **青之羽毛（青い羽根）**：愛情度達 50,000（粉色花苞）以上後，第 2 年起雜貨店開始販售。向對象出示即可求婚。
- **雙人床（ダブルベッド）**：所有角色的求婚前置條件之一，需透過自宅增建取得。
- **婚禮時間**：求婚成功後一週舉行婚禮。若婚禮當天遇到節日或村民生日，婚禮延至翌日（兩村的節日皆有影響）。

### 艾瑞拉（アリエラ）

![艾瑞拉頭像](../images/leomoon173-pixnet-5010160702/ariela.jpg)

**居住地點**：藍鈴村公所（村役場）1 樓

**特殊規則**：
- 與賢者大人相同，花朵顏色在結婚後才會變化
- **艾瑞拉沒有戀愛事件**，但有登場事件
- **結婚後無法懷孕生子**

#### 登場事件

| 項目 | 內容 |
|------|------|
| 時間 | 第 2 年秋以後，22:00 以後 |
| 天氣 | 晴天 |
| 地點 | 從山道走到山頂 |
| **前置條件** | 羅萬（ロヴェン，神父）好友度 2 朵花以上 |

- `「…引き返そう」` / `「行ってみる」`
- 選項選錯下個月可重新觸發

#### 求婚條件（共 7 項，須**全部**達成）

1. 愛情度 65,000 以上（5 朵花＋大花）
2. 擁有雙人床
3. 隧道開通
4. 與艾瑞拉約會次數 **100 次以上**
5. 送艾瑞拉禮物 **150 次以上**
6. 冰淇淋（アイスクリーム）至少送過 **1 次**（黑雞蛋＋調味料製作，或咖啡店購買）
7. 蒐集 6 色耀珠，並向[[女神大人]]許願 `「アリエラと結婚させて！」`（需愛情度 55,000 以上才出現此選項）

**求婚方式**：達成所有條件後，任意日期、晴天或雪天，拿青之羽毛（青い羽根）給艾瑞拉。
