# 好感度與愛情度系統說明

本遊戲對所有主要村民都可以提升好感度（友好度）與愛情度。好感度以花朵數顯示於角色資訊頁面。可攻略戀愛對象的愛情度則以花朵「顏色」表示，與一般村民的「數量」系統略有不同。

## 花朵顏色等級與對應數值

| 花色 | 數值範圍 |
|------|----------|
| 白色花（白花） | 0 ～ 10,000 |
| 紫色花（紫花） | 10,001 ～ 20,000 |
| 藍色花（青花） | 20,001 ～ 30,000 |
| 綠色花（緑花） | 30,001 ～ 40,000 |
| 橙色花（橙花） | 40,001 ～ 50,000 |
| 粉色花苞（5 朵＋花苞） | 50,001 ～ 60,000 |
| 大紅花（5 朵＋大花） | 60,001 ～ 65,535 |

> 求婚需達到粉色花苞（50,001）以上，第 2 年起才能在雜貨店買到青之羽毛（青い羽根）。

> 迪魯卡（ディルカ）、米海爾（ミハイル）、賢者大人（賢者さま）、艾瑞拉（アリエラ）這 4 位角色的花朵，在結婚前不會隨愛情度變色。

## 每日互動獲得點數

- **每天對話**：＋100 點（一天限一次）
- **委託任務完成**：委託人 ＋50
- **配達任務完成**：委託人 ＋25、收件人 ＋25
- **節日參加**：約 ＋500～＋1,000

## 禮物送贈機制

禮物的加減分依物品對角色的好感等級而定。括號內數值為送給[[女神大人|女神大人（女神さま）]]時的特殊數值：

| 禮物類別 | 平日 | 生日 |
|----------|------|------|
| 最喜歡（大好き） | ＋800（＋900） | ＋4,000（＋4,000） |
| 喜歡（好き） | ＋300（＋250） | ＋2,000（＋750） |
| 普通 | ＋50（＋150） | ＋250（＋250） |
| 討厭（嫌い） | －300（－100） | －2,000（－1,000） |
| 最討厭（大嫌い） | －1,000（－900） | －5,000（－5,000） |

> 生日送最喜歡的禮物效益最大（＋4,000），送最討厭的禮物懲罰也最重（－5,000）。

## 約會事件系統

約會事件（デートイベント）在愛情度達 **5,000 以上（白色花上限）**後可觸發。

- **雨天可以約會**，節日當天無法約會
- 約會時地點隨機出現 3 個選項，依角色偏好不同：
  - 喜歡的地點：＋100
  - 一般地點：無變化
  - 討厭的地點：－100
- 各角色可約會的日期與時間請見個別角色條目（`date_days` 欄位）

## 保鮮度機制

禮物有 5 個保鮮度階段，影響送禮效果：

| 階段 | 日文原文 | 說明 |
|------|----------|------|
| 1 | `ぼつぐん！` | 鮮度拔群，效果最佳 |
| 2 | `まだまだイケる` | 還算可以 |
| 3 | `少しいたんだかな？` | 稍微有點壞了 |
| 4 | `そろそろヤバい…！` | **有點不妙** — 村民從此階段起拒絕收禮 |
| 5 | `くさってしまった…` | 已腐爛，完全不可使用 |

> 有完美保鮮馬車（パーフェクト保存車）可防止物品變質。

## 好感度/愛情度減少條件

以下行為會導致好感度下降：

- **長期不互動**：超過 **7 天**未與村民對話，有機率 －100
- **結婚伴侶**：超過 **5 天**未對話，有機率 －100
- **送討厭的禮物**：見上方禮物表
- **往特定地點丟棄物品**：見下方河流對應表

## 攻略建議

1. **每天對話**是最穩定且零成本的好感度累積方式，不要忘記。
2. **生日送最喜歡的禮物**效益最高（＋4,000），平時可優先準備。
3. 確認角色的 `loves` 禮物後批量製作備用，農作物、魚獲尤其方便。
4. 避免往河流或水田丟棄物品，以免影響多位村民好感。
5. **送禮前確認保鮮度**，`そろそろヤバい…！` 以後的物品村民會拒絕。
6. 約會地點選喜歡的，可額外獲得 ＋100 加成。
