## 達成事件是什麼

遊戲會持續在背景記錄玩家的各種行為次數與里程碑，例如飼養動物的隻數、學會的食譜數、抓過的昆蟲種類等等。當某一項數值累積到門檻，就會反映在對應的稱號等級上。

達成事件大致涵蓋以下幾個判定面向：

| 事件類型 | 判定依據 |
|---|---|
| 動物飼養 | 飼養動物的累積數量 |
| 食譜學習 | 習得食譜的累積數量 |
| 撿拾物品 | 在山裡撿拾物品的累積次數 |
| 捕魚（次數） | 同一魚種的累積釣獲次數 |
| 捕魚（種類） | 累積捕獲過的魚種數量 |
| 捕蟲（次數） | 同一昆蟲的累積捕捉次數 |
| 捕蟲（種類） | 累積捕獲過的昆蟲種數 |
| 作物收穫（次數） | 同一作物的累積收穫次數 |
| 作物收穫（種類） | 累積收穫過的作物種數 |
| 徒手抓魚 | 累積徒手抓魚次數 |
| 服裝收集 | 收集齊全套服裝 |
| 製造機／水車加工 | 累積加工完成次數 |
| 任務委託 | 委託任務等級提升（詳見 [[委託任務系統]]） |
| 騎馬移動 | 累積騎馬移動步數 |
| 山裡設施使用 | 累積使用次數 |
| 寵物飼養 | 特定寵物的飼養數量 |
| 女神祈願 | 累積投入女神泉的物品次數 |
| 耀珠收集 | 集齊特定數量的耀珠 |
| 主角暈倒 | 累積暈倒次數 |

這些累積數值大部分都會反映到下面各類「稱號」的等級門檻上。

## 稱號分類總覽

遊戲的稱號分成七大類，其中「任務稱號（使者等級）」已整理在 [[委託任務系統]] 的 LV1-10 對照表裡，這裡不重複列出，以下整理其餘六類。
