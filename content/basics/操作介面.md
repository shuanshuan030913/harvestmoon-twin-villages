# 操作介面

## 按鍵說明

| 按鍵 | 功能 |
|------|------|
| A | 確定、撿東西、使用工具 |
| B | 取消、跳躍、加快文字速度 |
| X | 打開背包 |
| Y | 將物品放進背包 |
| L | 跑步、翻頁 |
| R | 打開工具欄、翻頁、查看人物圖片 |
| SELECT | 地圖切換 |
| 觸控筆 | 觸碰下屏幕、操作任務告示牌 |

**移動技巧**：按住方向鍵的同時按 A 鍵，可自動朝該方向移動。

## 背包欄位

按 X 鍵開啟背包後，選取物品可執行以下操作：

| 指令（日文） | 動作 |
|---|---|
| そうぴ（裝備） | 裝備到手上 |
| もちあげ（手持） | 手持物品 |
| たぺろ（食用） | 直接吃掉以恢復體力 |
| おく（地面） | 放置到地面 |
| すてる（丟棄） | 丟棄 |

> 使用「おく」可以一次把重疊的資材全部放到地面，再用工具批量處理，效率較高。

## 下屏幕功能

下屏幕（DS 觸控螢幕）提供快速存取以下功能：

- **背包（かバん）**：管理持有物品
- **魚（さかな）**：已釣魚類紀錄
- **昆蟲（ムシカゴ）**：已捕昆蟲紀錄
- **圖鑑（ずかん）**：全圖鑑瀏覽（含 [[作物]]、[[魚]]、[[昆蟲]] 等）
- **任務委託（おつかいメモ）**：查閱與接受任務告示牌上的委託
- **地圖**：切換藍鈴村 / 此花村地圖
- **衣服更換**：更換主角外觀服裝

![下屏幕介面](../images/leomoon173-pixnet-5010107872/touch-screen.jpg)

## 村民識別

對話時，村民名字顏色代表性別：
- **紅色名字** → 女性
- **藍色名字** → 男性
