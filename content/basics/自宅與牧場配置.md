# 自宅與牧場配置

## 自宅設施一覽

| 位置 | 設施 | 說明 |
|------|------|------|
| 左側 | 日曆 | 顯示當前日期與季節 |
| 左側 | 收音機 | 需花 1500G 購買 |
| 左側 | 馬車裝飾 | 裝飾用 |
| 左側 | 倉庫 | 保鮮 1–2 週 |
| 左上 | 書櫃 | 查看資產表 |
| 左上 | 花籃 | 裝飾用 |
| 左上 | 床 | **存檔** |
| 左上 | 床頭櫃 | 放置獎座 |
| 右上 | 料理台 | 提供 2 種製作模式 |
| 右下 | 飼料箱 | 分小型（貓）/ 大型（狗）寵物 |
| 右下 | [[樹枝]]架 | [[貓頭鷹]]使用 |

> 存檔只能透過**睡覺（使用床）**進行，凌晨 5:00 強制暈倒不會自動存檔。
